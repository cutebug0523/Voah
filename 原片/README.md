# 原片目录

`原片/` 保存待入库或已入库的原始视频素材。Voah 的产品身份优先来自路径和人工输入，不靠模型硬猜。

## 推荐结构

```text
原片/{产品名}/
  {原始视频}.mp4
```

例如：

```text
原片/防晒气垫/
原片/气垫/
```

## 规则

- 一个单品尽量一个文件夹。
- 文件夹名就是默认产品线索，后续可映射到 `product_slug`。
- 不要在入库流程里改动原片；所有裁切、压缩、代理视频都放 `cache/`。
- 如果从微信、剪映或其他软件导入素材，先复制到产品文件夹，再入库。
- 原片带硬字幕是常态，字幕风险在入库理解和后续素材选择里处理。
